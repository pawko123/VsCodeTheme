# Andromeda TokyoNight VS Code Theme

A local Visual Studio Code theme that reuses the same palette you now have in the Andromeda Ã— TokyoNight GNOME Shell port.

## Install locally

1. Open VS Code and run the command **Developer: Install Extension from Location...**.
2. Select this folder:
   ```
   /home/pawkoserver/.vscode/andromeda-tokyonight-theme
   ```
3. After VS Code reloads, pick **Andromeda TokyoNight** from the Color Theme picker.

You can also install it via CLI:
```bash
code --install-extension /home/pawkoserver/.vscode/andromeda-tokyonight-theme
```
